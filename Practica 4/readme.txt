Este programa fue desarrollado en el lenguaje de programación python.
No se requiere ninguna biblioteca externa para su ejecución.
para correrlo, seguir los siguientes pasos:
	- Abrir un aterminal
	- Posicionarse en la ubicación del archivo .py
	- Ejecutar el comando "python p4.py"

Una vez ejecutado el programa, se le pedirá seleccionar una opción, la opción 1 es para ejecutar la solución utilizando 
algoritmo voraz y la opción 2 es para ejecutar la solución utilizando programación dinámica.

Una vez seleccionada la opción, se le pedirá ingresar los tamaños de las matrices a multiplicar. Por ejemplo, si se desean 
multiplicar las matrices A(20x5), B(5x10), C(10x20), deberá ingresar en el programa los tamaños de la siguiente manera: 
20x5,5x10,10x20.

El programa mostrará en pantalla el número mínimo de multiplicaciones.
